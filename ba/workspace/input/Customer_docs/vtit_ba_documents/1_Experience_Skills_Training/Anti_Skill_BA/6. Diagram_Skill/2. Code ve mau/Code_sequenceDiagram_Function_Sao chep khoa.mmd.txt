sequenceDiagram
    actor Admin as Quản trị viên
    participant UI as Admin UI
    participant SSO as SSO/Keycloak
    participant API as Backend TTHDL
    participant DB as CSDL TTHDL
    participant LOG as Nhật ký hệ thống

    Admin->>UI: Chọn khóa cần sao chép
    UI->>SSO: Kiểm tra quyền thao tác
    SSO-->>UI: Có quyền

    UI->>API: Lấy thông tin khóa nguồn
    API->>DB: Truy vấn khóa nguồn
    DB-->>API: Trả thông tin khóa nguồn

    API-->>UI: Tạo bản sao dữ liệu khóa
    UI-->>Admin: Hiển thị form bản sao

    Admin->>UI: Cập nhật tên khóa, giá trị khóa, thời hạn hiệu lực
    UI->>API: Gửi yêu cầu lưu khóa sao chép

    API->>API: Validate dữ liệu
    API->>DB: Kiểm tra trùng tên khóa / chồng lấn hiệu lực
    DB-->>API: Không trùng / không chồng lấn

    API->>API: Mã hóa / lưu an toàn giá trị khóa
    API->>DB: Lưu khóa mã hóa sao chép
    DB-->>API: Lưu thành công

    API->>LOG: Ghi lịch sử thao tác
    LOG-->>API: Thành công

    API-->>UI: Trả kết quả thành công
    UI-->>Admin: Thông báo thành công, làm mới danh sách
