# domain-knowledge/ — Kiến thức Miền Hàng không

## Mục đích

Lưu trữ tài liệu kiến thức miền hàng không dùng làm nền tảng phân tích nghiệp vụ: quy định, tiêu chuẩn, thuật ngữ, và tài liệu tham chiếu ngành.

**Tài liệu nguồn chỉ-đọc** — không chỉnh sửa; mọi trích dẫn/phân tích đưa vào `../drafts/phan-tich/`.

## Phạm vi nội dung

| Nhóm | Mô tả | Ví dụ |
|---|---|---|
| Quy định CAAV | Quy định của Cục Hàng không Việt Nam | Thông tư 03/2016/TT-BGTVT |
| Tiêu chuẩn ICAO | ICAO Annex, Doc, Circular | ICAO Annex 6, Doc 9868 |
| Tiêu chuẩn IATA | OCC, ground handling, dispatch | IATA OCC Manual |
| FTL / Fatigue | Quy định giờ bay và nghỉ ngơi tổ bay | EASA FTL, CAAV FTL |
| Thuật ngữ | Glossary, từ điển hàng không | ICAO Doc 9713 |
| Reference | Tài liệu tham chiếu ngành khác | Airline OCC SOPs |

## Quy ước đặt tên

- Theo chuẩn: `<to-chuc>-<loai>-<mo-ta>.md` — ví dụ: `icao-annex6-flight-operations.md`
- File PDF/gốc: giữ nguyên tên gốc
