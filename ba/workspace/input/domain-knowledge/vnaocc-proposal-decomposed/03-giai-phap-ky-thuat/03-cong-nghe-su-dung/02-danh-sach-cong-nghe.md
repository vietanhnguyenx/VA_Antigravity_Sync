## Nội dung trích nguyên văn (Verbatim Extract)

| Công nghệ | Mô tả |
|---|---|
| Kubernetes (K8s) | Nền tảng nguồn mở để quản lý các ứng dụng đóng gói và service, điều phối microservices cho hệ thống. |
| CI/CD | Quy trình tích hợp và chuyển giao liên tục giúp phát triển phần mềm nhanh và đúng tiến độ. |
| LTMG (Loki, Tempo, Mimir, Grafana) | Hệ thống giám sát (Monitoring) và cảnh báo tập trung cho toàn bộ hệ thống. |
| Angular | Framework frontend do Google phát triển, hỗ trợ xây dựng ứng dụng web với kiến trúc module, khả năng mở rộng cao |
| Flutter | Ngôn ngữ lập trình phát triển ứng dụng di động do Google phát triển được sử dụng rộng rãi. |
| Keycloak | Giải pháp Single Sign-On (SSO) để quản lý danh tính và truy cập tập trung. |
| Apache Kafka | Hệ thống Message Broker pub/sub phân tán để trung chuyển dữ liệu thời gian thực. |
| WebSocket hoặc SSE | Công nghệ hỗ trợ giao tiếp real-time giữa client và server. |
| Nginx Gzip Compression | Phương pháp nén dữ liệu tại server trước khi gửi đến client. |
