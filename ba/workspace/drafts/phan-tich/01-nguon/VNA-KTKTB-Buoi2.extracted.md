Kết quả khảo sát buổi 2 VNA - Phòng Kỹ thuật khai thác bay

1. Nhóm phần mềm biên soạn (tài liệu MEL)

- Tìm hiểu quy trình và cách sử dụng phần mềm:

+ Hiểu quy trình soạn / chỉnh sửa tài liệu trên Boeing (FrameMaker Adobe + IXP)

+ Hiểu quy trình soạn / chỉnh sửa tài liệu trên AirBus (FODM 10.8 và)

- VTIT trình bày ý tưởng sơ bộ về tool cho khách hàng

+ Tool soạn thảo tài liệu hỗ trợ cho cả Boeing và Airbus (Khách hàng rất thích ý này)

+ Hỗ trợ revision tài liệu, để xem được sự thay đổi chỉnh sửa

+ Áp dụng quản lý trạng thái bộ tài liệu (Airbus-Working-Pending-Airline)

+ Hỗ trợ hiển thị compare các version tài liệu (Khách hàng rất thích ý này)

+ Xuất được tài liệu ra các định dạng phù hợp với máy bay Airbus và Boeing (XML, OPS và 1 định dạng nữa)

+ Riêng dòng Airbus350 thì cần phải có thẻ kí mình cũng đề xuất là riêng bước kí sẽ không làm được trực tiếp thì vẫn tiếp tục thủ công dùng FODM (Khách hàng đồng ý)

+ Mỗi đợt hãng máy bay xuất bản hotfix (SB) thì KH mong muốn công cụ của mình cũng có thể hỗ trợ so sánh và áp dụng nhanh các lần hotfix vào đúng tài liệu, hỗ trợ áp dụng với loại số hiệu tàu bay

+ Tool có thể phân loại và áp dụng riêng tài liệu cho từng loại tàu bay, số hiệu tàu bay

=> Khách hàng hài lòng với đề xuất ý tưởng của VTIT.

- Công việc tiếp theo cần nguyên cứu tính khả thi

+ Đọc / sửa file định dạng XML trên Frame maker

+ Đọc / sửa file định dạng XML trên FODM

+ Nghiên cứu cách xuất file qua các định dạng XML, OPS.

2. Tool tính toán cân bằng trọng tải

- Tìm hiểu quy trình và cách sử dụng phần mềm:

+ Hiểu quy trình khai báo dữ liệu, tính toán ra cân bằng tải của máy bay.

+ Hiểu đầu ra tài liệu

- VTIT trình bày ý tưởng sơ bộ về tool cho khách hàng

+ Tool sẽ cho phép khai báo dữ liệu cố định theo loại tàu bay và số hiệu tàu bay 		+ Tool sẽ cho phép Import dữ liệu tàu bay từ Nhà sản xuất (format .wbd)

+ Tool sẽ áp dụng các công thức tính toán cân bằng trọng tải từ file excel để tính toán ra số liệu DOW, DOI, DOCG của tàu bay

+ Tool sẽ export ra file Word theo template, Excel, PDF

=> Khách hàng hài lòng với đề xuất ý tưởng của VTIT.

- Công việc tiếp theo cần nguyên cứu tính khả thi

+ Đọc / tìm hiểu các trường dữ liệu cần khai báo

+ Phân tích file excel để lấy về các công thức tính toán 		+ Cần phân tích thêm file .wbd từ nhà sản xuất máy bay

+ KH có nêu nhu cầu đấu nối với dự án MO Plus. (Chưa rõ tính năng nào trong MO Plus)
